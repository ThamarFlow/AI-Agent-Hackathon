# Internal Docs Q&A Agent for Teams (Built using Chatbase)

This is a smart internal chatbot designed to help teams quickly access important documentation without wasting time searching across multiple platforms.

## What it does
It allows employees to ask questions in natural language like “What’s our refund policy?” or “How to request design assets?” — and instantly get answers from internal team documents such as Notion, Google Docs, Confluence, etc.

## Built With
- Chatbase (No-code AI Agent Builder)
- HTML (for embedding into web apps)

## How to Use
Just open `index.html` in a browser or deploy it online using Netlify or GitHub Pages.

You can embed it on internal tools or platforms like Slack using the Chatbase embed code.
